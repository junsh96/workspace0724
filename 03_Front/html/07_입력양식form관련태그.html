폼 관련태그

form 태그는 사용자로부터 입력을 받아 서버에 전달하는 영역을 정하는 html태그
form 태그내에 submit버튼을 클릭시 해당 form안에 작성된 input요소의 값들을 서버에 전달. - action : 해당 form에 사용자가 입력한 값들을 전달하면서 요청을 보낼 서버의 경로
- method : 요청 전송방식을 지정하는 속성(get(기본값),post)
get > 요청시 사용자가 입력한 값들을 head에 담아 전송한다. -> url에 값이 전부 노출
post > 요청시 사용자가 입력한 값들을 http 본문(body)에 숨겨서 전송 -> url에 값이 노출x
form내에 submit버튼을 클릭시 action에 지정된 서버로 사용자가 입력한 값들이 전달되면서 요청됨.



로그인


text와 관련된 input태그의 타입

type="text"

- 한줄짜리 텍스트를 입력할수있는 텍스트 상자
- maxlength(최대길이),placeholder(설명),required(필수값)

아이디 :

type="password"

- 입력한 문자를 *** 형태로 가려줌. 비밀번호 입력에 사용. 속성은 text와 동일

이외 텍스트 관련 타입

여러줄 입력 : textarea

- 여러 줄의 텍스트를 한번에 입력 가능
- cols, rows로 크기 지정. resize: none으로 크기조절 제한 가능

숫자와 관련된 input태그의 타입

type = "number"

-기본적으로 숫자만 입력, 스핀 박스 표시
-min,max,step등의 속성 사용 가능

type="range"

슬라이드바를 통해 숫자를 입력.

날짜 및 시간 입력
