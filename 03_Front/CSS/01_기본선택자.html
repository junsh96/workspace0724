기본 선택자

선택자란

특정 html을 선택하고자 할때 사용하는 기능.
해당요소를 선택해서 원하는 스타일과 기능을 적용할수있다.

1. 전체 선택자 : *

현재 이 문서 (DOM Tree)상의 모든 요소들을 다 선택하고자 할 때

        *{
            스타일 속성 : 값 *
            스타일 속성 : 값 * 
        }

2.태그 선택자 : 태그명

현재 이 문서상 해당하는 태그들을 전부 선택하고자 할때 

        태그명 {
            스타일 속성 : 값 *
            스타일 속성 : 값 *  
        }

3. ID 선택자 : #ID

id 속성 : 해당문서에 딱 하나만 가지고있는 고유 식별 속성
문서상에서 특정 html요소를 하나만 선택하고자 할때 사용.

  1. 리스트 출력
  2. 리스트 출력
  3. 리스트 출력
  4. 리스트 출력

4. 클래스 선택자 : .class명

class속성 : 특정 요소들을 묶어서 가져오기 위해 부여하는 식별자(그룹 식별)
문서상에서 내가 원하는 요소여러개를 선택하고자 할때 사용.

  • 클래스 0
  • 클래스 x
  • 클래스 0
  • 클래스 0
